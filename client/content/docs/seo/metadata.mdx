# Metadata Standard

Every destination page requires:

- Unique title
- Unique meta description
- Canonical URL
- Locale
- Alternate-language URL
- Open Graph title
- Open Graph description
- Open Graph image
- Last updated
- Last verified

## Suggested URL Pattern

```text
/id/bali/tanah-lot
/en/bali/tanah-lot
```
