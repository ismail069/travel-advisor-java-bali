# Schema Reference

Use structured data only when it matches visible page content.

Recommended types:

- `TouristAttraction`
- `Place`
- `GeoCoordinates`
- `BreadcrumbList`
- `FAQPage`
- `ImageObject`

Do not include hidden FAQ content in `FAQPage`.
