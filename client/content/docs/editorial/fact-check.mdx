# Fact-check Guide

## Minimum Sources

1. Official operator or attraction
2. Government or official tourism authority
3. Independent operational source

## Must Be Rechecked Before Publication

- Ticket price
- Opening hours
- Access rules
- Temporary closures
- Transport schedule
- Parking
- Accessibility
- Facilities
- Performance schedule

Every changing value requires a `lastVerified` date.
