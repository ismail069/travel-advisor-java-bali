# Introduction

Travel Advisor Java & Bali is a bilingual destination-content website focused on practical, trustworthy, and maintainable travel guides.

The project prioritizes content quality over platform complexity.

## Core Principles

- Write for travelers, not for keyword density.
- Never copy another publication.
- Never invent operational facts.
- Use at least three reliable sources.
- Mark changing information with a verification date.
- Write Indonesian and English naturally.
