# Publishing Checklist

## Editorial

- [ ] Original writing
- [ ] Indonesian reviewed
- [ ] English reviewed
- [ ] No unsupported superlatives
- [ ] No fabricated experience

## Fact-check

- [ ] Minimum three sources
- [ ] Ticket verified
- [ ] Opening hours verified
- [ ] Access rules verified
- [ ] Facilities reviewed
- [ ] Accessibility claims reviewed
- [ ] Last verified added

## SEO

- [ ] Unique title
- [ ] Unique meta description
- [ ] Canonical URL
- [ ] Alternate-language URL
- [ ] Internal links
- [ ] Schema validated

## Images

- [ ] Legal source
- [ ] License recorded
- [ ] Attribution added if required
- [ ] Alt text written
