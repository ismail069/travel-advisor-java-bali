# Glossary

## Canonical Destination Template

The required section order used by every destination page.

## Dynamic Information

Information likely to change, including tickets, hours, access rules, and facilities.

## Golden Reference Batch

The five Bali destination pages used as the final quality benchmark.

## Last Verified

The date on which a changing fact was checked against a reliable source.

## MDX

Markdown extended with reusable UI components.

## Source of Truth

The canonical documentation that controls all production decisions.
